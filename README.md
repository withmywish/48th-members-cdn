# 48TH Members CDN

Content Delivery Network (CDN) for BNK48 and CGM48 member photos, powered by jsdelivr. This repository serves as a centralized asset hub for developers building fan-made websites.

&nbsp;

## How to use

### Optimized Version: WebP format with 250x250px

`https://cdn.jsdelivr.net/gh/withmywish/48th-members-cdn@[Version Number]/optimized/[Member Name in Lowercase].webp`

e.g. https://cdn.jsdelivr.net/gh/withmywish/48th-members-cdn@3/optimized/emma.webp

### Original Version: JPG format

`https://cdn.jsdelivr.net/gh/withmywish/48th-members-cdn@[Version Number]/original/[Member Name in Lowercase].jpg`

e.g. https://cdn.jsdelivr.net/gh/withmywish/48th-members-cdn@3/original/emma.jpg
